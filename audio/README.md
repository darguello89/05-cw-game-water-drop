Place your downloaded sound effects in this folder using these exact filenames:

- button.mp3  -> played when Start/Reset is clicked
- collect.mp3 -> played when a water can is collected
- miss.mp3    -> played when a water can is missed
- win.mp3     -> played when the player wins

Free sources you can use:
- https://pixabay.com/sound-effects/
- https://freesound.org/

Note: The game includes generated fallback tones if these files are missing, so gameplay still has sound feedback during development.
